# FreakyTravellers

An AI-powered budget travel optimization web app. Enter your source, destination, and budget — the platform finds the cheapest transport, best hotels within your price range, and recommends restaurants, all in one place.

## Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion, Zustand
- **Backend**: Node.js, Express.js, MongoDB
- **Auth**: JWT + Email OTP verification
- **Payments**: Razorpay

## Features

- Smart budget allocation across transport, hotels, and food
- Cheapest transport selection (bus/train/flight)
- Hotel and restaurant recommendations within budget
- Secure authentication with email OTP
- Razorpay payment integration
- Responsive UI with animations

## Getting Started

```bash
# Backend
cd server && npm install && npm run dev

# Frontend
cd client && npm install && npm run dev
```

Set up `.env` in `server/` and `.env.local` in `client/` with your credentials before running.
